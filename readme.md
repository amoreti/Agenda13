# 🎬 Meu Portal de Entretenimento Digital

✨ Um projeto desenvolvido com HTML e CSS com o objetivo de praticar conceitos básicos de estruturação de páginas web e estilização.

---

## 📌 Sobre o Projeto

Este projeto apresenta um portal simples de entretenimento digital, organizado em três seções principais:

- 🎥 Filmes e Séries  
- 🎧 Música e Podcasts  
- 🎮 Games e Tecnologia  

Cada seção contém imagens e textos descritivos sobre o tema.

---

## 🎨 Layout e Estilo

O design foi pensado com um tema moderno e contrastante:

- 🖤 Fundo principal preto  
- 💗 Detalhes em rosa (títulos e bordas)  
- 📦 Seções organizadas com Flexbox  

---

## 🚀 Tecnologias Utilizadas

- ✅ HTML5  
- ✅ CSS3  
- ✅ Flexbox  
- ✅ CSS Reset  

---
